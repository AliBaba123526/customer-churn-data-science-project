# Customer Churn Prediction | End-to-End Data Science Project

## Project Overview
This project is an end-to-end data science project focused on solving a real business problem: reducing customer churn.

The project follows a business-first approach:
- Define the problem before coding
- Focus on decision-making, not just models
- Apply structured analytical thinking
- Deliver actionable insights

## Business Problem
Customer churn is a major issue for businesses because losing customers reduces revenue.

The goal of this project is to identify customers who are likely to leave, so the business can take action to retain them.

## Project Approach
This project is built using a structured data science workflow:

1. Define the problem (decision, action, outcome)
2. Ask analytical questions
3. Clean data with purpose
4. Perform EDA with intent
5. Build machine learning models with logic
6. Evaluate results critically
7. Provide business insights

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Status
Project setup in progress. Development will follow a step-by-step approach with strong focus on reasoning and explanation.
